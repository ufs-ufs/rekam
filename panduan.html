1. Pendahuluan

Selamat datang di Q-Stat (Quick Statistics), aplikasi web berbasis HTML dan JavaScript yang dirancang untuk membantu Anda melakukan analisis statistik dasar dengan cepat dan mudah diakses melalui web. Q-Stat menyediakan berbagai fitur seperti analisis deskriptif, uji statistik, visualisasi data, dan ekspor hasil analisis, mirip dengan perangkat lunak statistik profesional seperti SPSS.

2. Persyaratan Sistem

Untuk menggunakan Q-Stat, pastikan perangkat Anda memenuhi persyaratan berikut:

  • Browser: Google Chrome, Mozilla Firefox, Microsoft Edge, atau Safari (versi terbaru).
  • Koneksi Internet: Stabil untuk memuat dependensi dan fitur ekspor PDF.
  • File Data: File CSV dengan delimiter tab (\t) atau koma (,).

3. Memulai Penggunaan

  1. Buka Aplikasi:
    • Buka file https://ufs-ufs.github.io/rekam/Q-Stat.html di browser pilihan Anda.
    • Anda akan disambut oleh antarmuka pengguna yang bersih dan intuitif.
  2. Navigasi Utama:
    • Navbar: Terletak di bagian atas, menampilkan judul aplikasi.
    • Bagian Utama: Terdiri dari berbagai kartu (cards) yang mengelompokkan fitur-fitur utama seperti unggah data, pratinjau data, analisis, uji statistik, visualisasi, dan ekspor hasil.

4. Mengunggah File CSV

  1. Pilih File:
    • Klik tombol "Unggah File CSV".
    • Pilih file CSV yang ingin Anda analisis dari perangkat Anda.
  2. Pengaturan Encoding:
    • Centang kotak "Encode variabel kategorikal menjadi numerik" jika Anda ingin mengubah data kategorikal menjadi numerik untuk analisis statistik yang lebih lanjut.
  3. Format File:
    • Pastikan file CSV memiliki header yang jelas dan konsisten.
    • Contoh format CSV dapat ditemukan di bagian Pedoman Pemilihan Uji Statistik.

5. Pratinjau Data

Setelah mengunggah file CSV, data Anda akan ditampilkan dalam tabel interaktif menggunakan DataTables. Anda dapat melakukan:

  • Sorting: Klik pada header kolom untuk mengurutkan data.
  • Searching: Gunakan fitur pencarian untuk menemukan data spesifik.
  • Paging: Navigasi antar halaman data jika dataset besar.

6. Analisis Deskriptif

Q-Stat menyediakan analisis deskriptif dasar untuk setiap kolom numerik dalam dataset Anda, termasuk:

  • Mean (Rata-rata)
  • Median
  • Varians
  • Standar Deviasi
  • Min (Minimum)
  • Max (Maksimum)

Cara Menggunakan:

  1. Navigasi ke "Analisis Deskriptif":
    • Klik pada kartu "Analisis Deskriptif".
  2. Lihat Hasil:
    • Hasil analisis akan ditampilkan secara otomatis berdasarkan data yang diunggah.

7. Melakukan Uji Statistik

Q-Stat mendukung berbagai uji statistik dasar. Berikut adalah panduan untuk masing-masing uji:

a. T-Test

Kapan Menggunakan: Membandingkan rata-rata dua kelompok independen.

Langkah-langkah:

  1. Pilih "T-Test" dari Dropdown Uji Statistik.
  2. Pilih Grup 1 dan Grup 2:
    • Pilih dua variabel numerik yang ingin dibandingkan.
  3. Jalankan Uji:
    • Klik tombol "Jalankan Uji".
  4. Lihat Hasil:
    • Hasil uji T-Test akan ditampilkan dengan nilai t-statistik, p-value, confidence interval, dan hasil Levene's Test.

b. Korelasi Pearson

Kapan Menggunakan: Mengukur hubungan linier antara dua variabel numerik.

Langkah-langkah:

  1. Pilih "Korelasi Pearson" dari Dropdown Uji Statistik.
  2. Pilih Variabel X dan Variabel Y:
    • Pilih dua variabel numerik yang ingin dianalisis.
  3. Jalankan Uji:
    • Klik tombol "Jalankan Uji".
  4. Lihat Hasil:
    • Hasil korelasi akan ditampilkan dengan koefisien korelasi Pearson dan p-value.

c. ANOVA

Kapan Menggunakan: Membandingkan rata-rata lebih dari dua kelompok.

Langkah-langkah:

  1. Pilih "ANOVA" dari Dropdown Uji Statistik.
  2. Pilih Grup:
    • Pilih minimal dua variabel numerik yang ingin dibandingkan.
  3. Jalankan Uji:
    • Klik tombol "Jalankan Uji".
  4. Lihat Hasil:
    • Hasil ANOVA akan ditampilkan dengan nilai F-statistik, p-value, dan hasil Levene's Test.

d. Regresi

Kapan Menggunakan: Memprediksi nilai variabel dependen berdasarkan variabel independen.

Langkah-langkah:

  1. Pilih "Regresi" dari Dropdown Uji Statistik.
  2. Pilih Variabel Independen (X) dan Variabel Dependen (Y):
    • Pilih satu variabel numerik sebagai prediktor dan satu variabel numerik sebagai target.
  3. Jalankan Uji:
    • Klik tombol "Jalankan Uji".
  4. Lihat Hasil:
    • Hasil regresi akan ditampilkan dengan slope, intercept, R-squared, p-value, dan confidence interval slope.

e. Chi-Square

Kapan Menggunakan: Menguji asosiasi antara dua variabel kategorikal.

Langkah-langkah:

  1. Pilih "Chi-Square" dari Dropdown Uji Statistik.
  2. Pilih Variabel X dan Variabel Y:
    • Pilih dua variabel kategorikal yang ingin dianalisis.
  3. Jalankan Uji:
    • Klik tombol "Jalankan Uji".
  4. Lihat Hasil:
    • Hasil Chi-Square akan ditampilkan dengan nilai statistik Chi-Square, degrees of freedom, dan p-value.

8. Visualisasi Data

Q-Stat menyediakan berbagai tipe grafik untuk memvisualisasikan data Anda:

  • Diagram Batang (Bar Chart)
  • Diagram Garis (Line Chart)
  • Diagram Pie (Pie Chart)
  • Histogram
  • Scatter Plot

Cara Menggunakan:

  1. Pilih Tipe Grafik:
    • Pilih jenis grafik yang diinginkan dari dropdown "Tipe Grafik".
  2. Pilih Kolom:
    • Pilih kolom data yang ingin divisualisasikan.
  3. Untuk Scatter Plot:
    • Pilih juga variabel Y untuk grafik Scatter Plot.
  4. Generate Grafik:
    • Klik tombol "Generate Grafik" untuk membuat dan menampilkan grafik.
  5. Interaksi dengan Grafik:
    • Grafik yang dihasilkan interaktif dan responsif terhadap ukuran layar.

9. Mengekspor Hasil

Q-Stat memungkinkan Anda untuk mengekspor data dan hasil analisis dalam format CSV dan PDF.

a. Ekspor ke CSV

Langkah-langkah:

  1. Klik Tombol "Ekspor CSV":
    • Terletak di bagian "Ekspor Data".
  2. Download File:
    • File CSV akan diunduh secara otomatis ke perangkat Anda.

b. Ekspor ke PDF

Langkah-langkah:

  1. Klik Tombol "Ekspor PDF":
    • Terletak di bagian "Ekspor Data".
  2. Download File:
    • Laporan lengkap dengan tabel dan grafik akan diunduh dalam format PDF.

10. Mapping Encoding Variabel Kategorikal

Jika Anda memilih opsi "Encode variabel kategorikal menjadi numerik", Q-Stat akan mengubah data kategorikal menjadi numerik untuk analisis statistik yang lebih lanjut.

Cara Menggunakan:

  1. Aktifkan Encoding:
    • Centang kotak "Encode variabel kategorikal menjadi numerik" sebelum mengunggah file CSV.
  2. Lihat Mapping:
    • Setelah data diunggah dan diproses, mapping encoding akan ditampilkan dalam modal.
  3. Contoh Mapping:
    • Jenis Kelamin:
      • Laki-laki: 0
      • Perempuan: 1

11. Rangkuman dan Laporan

Q-Stat menyediakan fitur "Laporan/Rangkuman Hasil Analisis" yang menampilkan ringkasan hasil analisis statistik yang telah dilakukan.

Cara Menggunakan:

  1. Lihat Rangkuman:
    • Setelah melakukan uji statistik, hasil dan penjelasan akan ditambahkan ke daftar rangkuman.
  2. Navigasi Rangkuman:
    • Klik pada item rangkuman untuk melihat penjelasan mendetail tentang hasil uji statistik.

14. Tips dan Trik

Persiapkan Data dengan Baik:

  • Pastikan file CSV memiliki header yang jelas dan konsisten.
  • Hindari nilai kosong atau tidak sesuai format dalam dataset.

Gunakan Encoding untuk Variabel Kategorikal:

  • Aktifkan opsi encoding untuk mengubah data kategorikal menjadi numerik agar dapat dianalisis lebih lanjut.

Periksa Asumsi Statistik:

  • Sebelum menjalankan uji statistik, pastikan data memenuhi asumsi normalitas dan homogenitas varians.

Manfaatkan Visualisasi:

  • Gunakan berbagai tipe grafik untuk memahami distribusi dan hubungan antar variabel dalam data Anda.

Ekspor Hasil Secara Berkala:

  • Simpan hasil analisis Anda secara berkala dalam format CSV atau PDF untuk dokumentasi dan presentasi.

15. Pertanyaan Umum (FAQ)

Q1: Apa saja tipe file yang dapat diunggah ke Q-Stat?
A1: Q-Stat mendukung file CSV dengan delimiter tab (\t) atau koma (,).
Q2: Bagaimana cara mengatasi error saat mengunggah file CSV?
A2: Pastikan file CSV memiliki header yang unik dan tidak ada nilai kosong. Periksa kembali format delimiter dan struktur data.
Q3: Apakah Q-Stat dapat menangani dataset besar?
A3: Q-Stat bergantung pada pemrosesan di sisi klien, sehingga kemampuan menangani dataset besar terbatas. Untuk dataset besar, pertimbangkan menggunakan perangkat lunak statistik profesional.
Q4: Bagaimana cara mengubah kembali data yang telah di-encode?
A4: Setelah meng-encode variabel kategorikal, Anda dapat melihat mapping encoding di modal "Mapping Encoding Variabel Kategorikal".
Q5: Apakah hasil analisis Q-Stat dapat diverifikasi dengan SPSS?
A5: Hasil dasar seperti T-Test, Korelasi, ANOVA, Regresi, dan Chi-Square seharusnya konsisten dengan SPSS, tetapi selalu disarankan untuk memverifikasi hasil penting menggunakan perangkat lunak statistik profesional.
